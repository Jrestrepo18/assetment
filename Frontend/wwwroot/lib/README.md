# lib

Esta carpeta contiene librerías de terceros como Bootstrap, jQuery, etc.

En este proyecto utilizamos CDNs para las librerías principales:
- Bootstrap 5.3.2
- Bootstrap Icons 1.11.1
- jQuery 3.7.1
- jQuery Validation

Si desea usar versiones locales, puede descargarlas aquí usando:
```bash
npm install bootstrap bootstrap-icons jquery jquery-validation
```
